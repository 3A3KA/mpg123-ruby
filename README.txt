= mpg123_ruby

http://github.com/slowjud/mpg123-ruby

== WARNING

This project is not even alpha. Its far from complete.

== DESCRIPTION:

Mpeg123_ruby is a wrapper around libmpeg123. See http://www.mpg123.de for more details on mpg123.

* Source code {hosted at GitHub}[http://github.com/slowjud/mpg123-ruby]
* Please {report any issues}[http://github.com/slowjud/mpg123-ruby/issues] on GitHub

== FEATURES/PROBLEMS:

* There are no features and everything is a bug.
* TODO: wrap error codes in exceptions and design a functional interface

== SYNOPSIS:

* Create an initialiser.
  MPG123::API.mpg123_init

== REQUIREMENTS:

Mpg123_ruby depends on mpg123 1.10 or higher. You can get the latest mpg123 from http://www.mpg123.de

== INSTALL:

  $ [sudo] gem install mpg123_ruby

== DEVELOPERS:

After checking out the source, run:

  $ rake newb

This task will install any missing dependencies, run the tests/specs,
and generate the RDoc.
